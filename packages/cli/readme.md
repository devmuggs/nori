# Nori CLI

Nori CLI is a command-line interface tool for interacting with the Nori platform. It allows users to manage their Nori projects, deploy applications, and perform various tasks directly from the terminal.

## Usage (npx)

-   **init**: Initialize a new Nori project.

    ```bash
    npx nori init

    args:
      --name <project-name>      Specify the name of the project.
      --output <directory>        Specify the output directory for the project.
    ```

-   **deploy**: Deploy your Nori application.
    ```bash
      npx nori deploy
    ```
