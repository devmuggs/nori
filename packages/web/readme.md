# Nori Web App

The Nori Web App is a user-friendly means of interacting with the nori platform through a web browser instead of the command line interface. It provides an intuitive graphical interface for managing projects, configuring settings, and monitoring tasks.

## Features

### MVP

- **Load/Save nori.yaml**: Load and save the nori.yaml configuration file.
- **Non-Technical User Friendly**: Designed for users who may not be comfortable with command line interfaces.
    - **Create a Collection**: Easily create new collections through the web interface.
    - **Collection Entry Management**: Add, edit, and delete entries within collections.
- **Project Management**: Manage multiple projects with ease.
- **Create New Project**: Start new projects directly from the web app.
- **Daemon Integration**: Seamlessly connect with the Nori Daemon for enhanced functionality. (hot reloading)

### Planned Features

- **LAN Discovery of Nori Daemon**: Automatically discover Nori Daemon instances on the local network for seamless connectivity.
- **User Authentication**: Secure access to the web app with user authentication features.
